🏥 Hospital Appointment System (React)

A modern and simple hospital appointment booking web application built using React + Vite.
Users can browse doctors and book appointments easily.

---

🚀 Features

- 🏠 Home page navigation
- 👨‍⚕️ Doctors listing page
- 📅 Appointment booking page
- ⚡ Fast performance with Vite
- 🔀 Routing using React Router

---

🛠️ Tech Stack

- React.js
- Vite
- React Router DOM
- CSS

---

📁 Project Structure

src/
 ├── components/
 ├── pages/
 │    ├── Home.jsx
 │    ├── Doctors.jsx
 │    ├── Appointment.jsx
 ├── App.jsx
 ├── main.jsx

---

⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/igsachin-001/hospital---appointment---react.git

2. Go to project folder

cd hospital---appointment---react

3. Install dependencies

npm install

4. Run the project

npm run dev
